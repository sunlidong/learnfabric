### event

# ec.RegisterChaincodeEvent() - 监听链码事件

# ec.RegisterTxStatusEvent() - 监听交易状态事件

# event.ClientOption - 通道事件客户端选项结构定义
# event.WithBlockEvents() - 创建监听区块事件的选项
# event.WithBlockNum() - 只监听指定编号的区块
